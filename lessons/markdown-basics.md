# יסודות Markdown

בשיעור הזה התלמידים לומדים איך טקסט פשוט הופך לדף לימודי ברור.

{: .note }
> Markdown מתאים במיוחד לכתיבת הנחיות קצרות, רשימות, טבלאות ודוגמאות קוד.

## מטרות

- לזהות כותרות, רשימות וקישורים.
- לכתוב טבלה קצרה.
- להבין למה קטעי קוד נשארים משמאל לימין.

## דוגמה לטבלה

| סימון | תוצאה | שימוש |
| --- | --- | --- |
| `#` | כותרת ראשית | פתיחת דף או נושא |
| `-` | רשימת נקודות | שלבי עבודה |
| `[טקסט](url)` | קישור | מעבר למקור נוסף |

## דוגמת קוד

```text
# Lesson title

- First task
- Second task
- Reflection
```

## משימה

כתבו פסקה קצרה על נושא שלמדתם השבוע, הוסיפו רשימת שלבים וקישור אחד פנימי אל [דף הבית]({{ '/' | relative_url }}).
